# event-registration

This is an in-progress project. Frontend is Angular, backend is a spring boot rest api. 

#Requirements
*Java JDK
*Tomcat
*MySQL
*NodeJS and npm

# Run
To run backend, open backend project, use gradle init->build->bootRun. 

To run frontend, 
*********
cd frontend
ng serve --proxy-config proxy.conf.json
*********
